# java-racingcar

## 기능 요구사항

---

- [ ] 경주에 참여할 자동차에 이름을 부여할 수 있다.
    - [ ] 중복하는 이름이 존재할 수 없다.
    - [ ] 이름은 최소 1글자 이상 5글자 이하이다.
- [ ] 자동차 게임을 진행할 횟수를 콘솔로 입력받는다.
- [ ] 0에서 9 사이의 랜덤한 값을 구한 후, 값이 4 이상이면 전진한다.
- [ ] 자동차 게임이 완료되면 우승자를 출력한다.
    - [ ] 우승자는 한 명 이상일 수 있다.
